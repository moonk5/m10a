# Description

['M10-A'](https://rama.works/m10-a/) is a programable macropad.
It is a custom keymap file for ['QMK'](https://github.com/qmk/qmk_firmware).  

# Compile & Flash Firmware

sudo make m10a:moonk5:dfu
